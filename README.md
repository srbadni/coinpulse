# CoinPulse

یک پروژه شروع تولیدی با Next.js App Router، TypeScript، Tailwind CSS، shadcn/ui و پشتیبانی کامل از رابط کاربری راست‌به‌چپ فارسی.

## اجرا

```bash
npm install
npm run dev
```

سپس آدرس `http://localhost:3000` را باز کنید.

## ویژگی‌ها

- Next.js 16 با App Router
- TypeScript با تنظیمات strict
- Tailwind CSS v4 و توکن‌های طراحی shadcn/ui
- shadcn/ui آماده استفاده با نمونه Button
- زبان فارسی، `dir="rtl"` و فونت Vazirmatn
- ساختار تمیز و قابل توسعه برای داشبورد یا SaaS
